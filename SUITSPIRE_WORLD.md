# 塔牌 / Suitspire — 世界设定 & 视觉参考

> 这份文档是给 3D 场景制作用的世界圣经。重点是**视觉、氛围、专有名词、关卡风格**，不写战斗数值。读完应能脑补出整个塔、四花色、五种族、NPC 形象与场景调子。

---

## 0. 一句话定调

**Suitspire** 是一座没有顶的塔。爬塔者带着一副 16 张的命运牌（四花色 ♠♦♥♣）从荒野出发，向上穿过哥布林洞窟、亡灵祭坛、暗影禁地，越爬越深。每三层有一个 Boss 守门人，每两关之间有一位等在路边的 NPC——商人、赌徒、神龛祭祀、戴乌鸦面具的术士、布满锈链的宝箱。

视觉调性：**暗调奇幻 + 卡牌图腾化 + 中式留白笔触感**。不是 Slay the Spire 那种纯粹西幻油画，更像**塔罗牌 × 山海经 × 阴暗童话**——花色是世界的物理法则，不是装饰。

---

## 1. 塔的结构

塔在视觉上**没有顶**。玩家爬到第 N 层都行，第 9 关之后的视觉主题会循环（暂时）。

塔身分四个主题段，每段有自己的色调与环境元素：

| 关卡 | 主题名 | 调性 | 主色 | 环境描述（一句话） |
|------|--------|------|------|---------------------|
| F1–F2 | **荒野边缘** | 黄昏旷野 | `#d4a64a`（琥珀金） | "塔影笼罩着第一片荒原，狼嚎在远处此起彼伏。" |
| F3–F5 | **哥布林洞窟** | 潮湿洞穴 | `#b87fd0`（紫罗兰） | "潮湿的洞穴回响着尖锐的笑声，火光摇曳。" |
| F6–F8 | **亡灵祭坛** | 寒蓝枯骨 | `#5b9fd8`（冷蓝） | "古老的咒语仍在回响，骨笛吟唱不息。" |
| F9+   | **暗影禁地** | 血红虚空 | `#ff5e5e`（血红） | "光明拒绝在这里停留，只剩低语。" |

**关卡布局**：单层是 Slay-the-Spire 式的纵向网格，从底部"起点 🚩"出发，经过 2–5 行中间节点，到达顶端"Boss 👑"。中间节点之间有分支，玩家选路径。

**节点类型**（每层都可能出现，各有图标 + 主色）：
- ⚔ **战斗节点** `#e63329`（红） — 常规战斗
- ✦ **精英节点** `#ff8c28`（橙） — 难度高、奖励好
- 👑 **Boss 节点** `#ffd460`（金） — 每 3 层顶端必有
- 🎲 **事件节点** `#9b59b6`（紫） — 随机 NPC 事件
- 💎 **商店节点** `#4db84d`（绿） — 流浪商人
- ⚒ **铁匠铺节点** `#f5c518`（黄） — 武器附魔

## 2. 四花色 ♠ ♦ ♥ ♣ — 世界的四种意志

四花色是塔牌世界的物理法则，不是图标。每种花色都是**一种攻击方式 + 一种生存哲学**。从 T1 到 T3 三档晋升，T3 解锁大招（消耗 10 点亲和）。

### ♠ 黑桃 / Spade — 战意
- **主色**：`#e6e6e6` 银白
- **气质**：钢铁、血怒、破甲
- **T1 锋锐怒涛 → T2 破甲狂攻 → T3 斩魂蓄势**
- **大招 · 狂战之击**：对目标造成其当前 HP 50% 的真实伤害（无视护甲）
- **视觉关键词**：刃光、白银、寒霜、断裂、骨与铁

### ♦ 方块 / Diamond — 灵巧
- **主色**：`#ff8a8a` 珊瑚粉
- **气质**：闪避、影分身、连击
- **T1 灵动闪步 → T2 灵巧连击 → T3 幻影成形**
- **大招 · 影舞步**：本回合敌人攻击全部闪避，下次攻击 hits × 3
- **视觉关键词**：粉色残影、半透明幻形、棱面切割、玫瑰金

### ♥ 红心 / Heart — 生命
- **主色**：`#ff5e5e` 血红
- **气质**：再生、吸血、绝境反扑
- **T1 生机涌动 → T2 绝境吸血 → T3 生命之泉**
- **大招 · 生命洪流**：HP 回满，永久 maxHP +5（整局限 3 次）
- **视觉关键词**：血脉、心跳、藤蔓、活体、温热

### ♣ 梅花 / Club — 庇护
- **主色**：`#aaffc3` 薄荷绿
- **气质**：防御、诅咒、群体压制
- **T1 魔法庇护 → T2 护体真言 → T3 禁咒蓄能**
- **大招 · 群体禁咒**：全体敌人 +3 沉默 +3 易伤 +3 中毒
- **视觉关键词**：符文圈、咒语、绿色磷火、根系、藤盾

## 3. 五种族 / 敌人图鉴

塔里所有敌人分五个种族。每种族有一种"灵魂碎片"，击杀掉落，铁匠铺用来附魔武器。

| 种族 | 中文 | 碎片名 | 图标 | 招牌能力 | 调性 |
|------|------|--------|------|----------|------|
| **beast** | 兽 | 兽魂 | 🐾 | 嗜血 | 野性、低位、群体 |
| **humanoid** | 人型 | 灵魂石 | 💎 | 战吼 | 武装、文明、组织 |
| **undead** | 不死 | 怨念 | 👻 | 不朽 | 枯骨、寒冷、低语 |
| **giant** | 巨怪 | 巨魂 | 🗿 | 重甲护体 | 山岳、缓慢、压迫 |
| **dark** | 暗影 | 暗影碎片 | 🌑 | 致命一击 | 虚空、瞬移、不可视 |

每种族有三个 tier：**普通 / 精英（elite）/ Boss**。下面列出全部敌人名（不会全部出现在一局，是个池子）。

### 3.1 兽 Beast — 荒野与洞窟的捕食者

**普通**：地鼠、野狼、毒蛇、蝙蝠、野猪、黑豹、狂狼、野熊、巨鼠、豺狗
**精英**：魔狼·首领、巨爪熊王、雷霆狼、毒蛇王、黑龙鳄、霜熊、影豹
**Boss**：**原初之兽 / 九尾狐妖 / 雷神之爪 / 兽王**

**攻击动作**：啃咬、撕咬、连咬、突袭、嚎叫、獠牙伤
**视觉**：毛发飞溅、利齿、低伏的姿态。Boss 级开始带元素（雷、霜）或多尾。
**典型场景**：F1–F5 出现最多

### 3.2 人型 Humanoid — 流浪武装与异端

**普通**：哥布林、强盗、盗贼、野蛮人、异端兵、佣兵、刺客、流浪者、猎人、异端徒
**精英**：盗贼头目、佣兵队长、黑暗骑士、异端祭司、野蛮酋长、杀手
**Boss**：**哥布林王 / 异端大主教 / 黑暗骑士团长 / 战团之王**

**攻击动作**：挥砍、连斩、突刺、重击、战吼、断筋、破甲
**视觉**：兵器明显、皮甲/锁甲、有阵营标志。哥布林王戴破王冠，黑暗骑士是黑色全身板甲，异端大主教穿带血符的法袍。
**典型场景**：F3–F5 主力，F6+ 配合不死

### 3.3 不死 Undead — 祭坛与古墓

**普通**：骷髅兵、丧尸、食尸鬼、白骨、墓守、腐尸、亡灵小兵、幽魂、干尸
**精英**：亡灵巫师、骨王、墓主、黑暗法师、鬼魅猎手、亡魂祭司
**Boss**：**巫妖 / 亡灵之主 / 骨之王座 / 死灵法师**

**攻击动作**：骨锤、骨爪、灵魂吸食、凋零术、诅咒、瘴气
**视觉**：枯骨、半透明灵体、冷蓝幽火、紫黑色咒文飘字。骨之王座是个长着骨刺的椅子本体（不是坐着的人）。
**典型场景**：F6–F8 主力

### 3.4 巨怪 Giant — 山岳与霜地

**普通**：小巨人、巨魔、山地野人、石头怪、沼泽巨魔、霜地野人
**精英**：石巨人、战争巨人、冰霜巨人、石神巨人、山岭哨兵
**Boss**：**山岳之主 / 时间巨灵 / 原始巨人 / 霜山巨王**

**攻击动作**：巨拳、双拳、跺地震、硬化、砸碎、重击
**视觉**：体型超出画面、有岩石/苔藓/冰晶贴皮、迟钝但每一击屏幕震屏。时间巨灵带沙漏元素，原始巨人裸身布满图腾刺青。
**典型场景**：F6+ 解锁

### 3.5 暗影 Dark — 禁地的低语者

**普通**：影爪、夜魔、深渊兽、暗影爪牙、黑暗使者、影魂
**精英**：暗影刺客、暗影王子、夜魔领主、深渊使者、影舞者
**Boss**：**黑龙 / 深渊领主 / 暗影之主 / 无相之主**

**攻击动作**：暗影斩、影矛连射、黑闪、暗杀重击、黑诅咒、腐血、暗影遁
**视觉**：边缘溶解、无固定形态、双眼是唯一可见的实体。无相之主**没有具象身体**，只是一团黑雾里浮着一张面具或一双眼睛。
**典型场景**：F9+ 解锁，是塔顶级别的存在

## 4. 楼层间 NPC & 事件（5 个）

每关末 70% 概率触发，从五个事件里等概率抽一个。每个事件都有自己的**场景位**——商人有摊位，赌徒有牌桌，神龛有石龛，术士有卷轴台，宝箱孤零零搁在路边。

### 4.1 🛒 流浪商人 / Wandering Merchant

> "一个穿着旧斗篷的商人正在摆摊，灵魂碎片是他认可的货币。"

- **形象**：旧斗篷遮脸（不要给五官），手指干瘦戴铜戒，斗篷下露出绑着小布包的腰带
- **摊位**：低矮木桌，上面摆 5 张卡片（背面朝上），旁边一盏油灯
- **氛围**：路边的临时摊，没有招牌，没有店面，背景是塔的内壁

### 4.2 🎲 赌徒 / Gambler

> "命运之轮在你面前转动。下注，看运气如何。"

- **形象**：戴宽檐帽的中年人，半边脸在阴影里，手里转着一枚硬币
- **摊位**：圆桌，上面有一个**命运之轮**（半人高的木轮，扇区涂红黑两色 + 金色边）+ 三摞筹码（小注/中注/大注）
- **氛围**：暧昧的灯光，桌面有酒渍

### 4.3 ⛲ 古老神龛 / Ancient Shrine

> "石龛上刻满血色符文。神祇要求献祭你的生命换取力量。"

- **形象**：没有 NPC，只有龛体本身
- **场景**：及腰高的方形石龛，正面刻满**血红色符文**（流着的，不是干涸的），龛顶是一个浅浅的圆盘（盛祭品用），后面的墙上有一个看不清面容的浮雕脸
- **氛围**：冷光从龛顶圆盘上方斜射下来，地上有干涸血迹

### 4.4 🐦‍⬛ 诡异术士 / Strange Sorcerer

> "一位戴着乌鸦面具的术士对你点头。她递出三张折叠的卷轴。"

- **形象**：**女性**，戴**乌鸦面具**（白色喙、黑色眼眶、有羽毛遮挡），深紫色长袍，手腕缠绷带
- **场景**：站在一张矮桌后面，桌上放三卷折好的羊皮卷（用蜡封口）
- **氛围**：诡异、礼貌、不说话只点头。背景有一只活的乌鸦栖在塔壁的烛台上

### 4.5 📦 神秘宝箱 / Mysterious Chest

> "锁链已经锈蚀。打开会获得财宝，也可能触发古老的陷阱。"

- **形象**：没有 NPC，纯静物
- **场景**：木箱，包铁角，**生锈的锁链缠了三圈**，锁头掉在地上像是早就坏了。箱缝里**透出一点点橙色光**（暗示里面有东西）
- **氛围**：路当中孤零零一个箱子，没有任何遮挡，**故意可疑**

## 5. 铁匠铺 ⚒ Forge（地图节点，不是事件）

> "⚒ 铁匠铺：使用灵魂碎片为武器附魔，或跳过。"

- **形象**：铁匠**不出现脸**（背对玩家或埋头打铁），围皮裙、肌肉手臂、戴单眼护目镜
- **场景**：石砌火炉 + 砧板 + 工具墙（挂着钳子、锤子、各种没见过的奇怪工具），地上散着木炭和碎铁
- **氛围**：温热橙光，铁锤敲击的回响。**有 25% 概率出现"5 折特惠"**，此时炉火变金色，炉子上方飘一条金色横幅
- **特殊**：附魔时，武器悬浮在砧板上方，符文从下往上流动注入

## 6. 武器与防具 — 玩家的装备词库

玩家的视觉装扮由当前装的武器/防具决定。**同款武器最多叠 4 张**（数值更强，外观可考虑添加华饰）。

### 武器（13 种）

| 中文 | 类型 | 视觉关键 |
|------|------|----------|
| **短剑** | 起手剑 | 普通铁剑，没装饰 |
| **长剑** | 标准剑 | 长一点、有十字护手 |
| **匕首** | 短刃 | 弯曲刀身、皮柄 |
| **血裂刃** | 邪剑 | 黑色剑身、刃上流着血色凹槽 |
| **战弓** | 远程 | 长弓 + 简单装饰 |
| **双刀** | 双持 | 两把短刀 |
| **巨锤** | 重武 | 方头锤 + 长柄 |
| **法杖** | 法系起手 | 木杖 + 顶端小水晶 |
| **链刃** | 软兵器 | 锁链连接刀身 |
| **狂剑** | 大剑 | 巨型双手剑、有缺口 |
| **法师杖** | 高阶法杖 | 杖头有发光的法球 + 星形装饰 |
| **连弩** | 多发远程 | 弩 + 上方箭匣 |
| **破军** | 穿甲名剑 | 形似细长大刀，刃口锯齿状，名字带军令气息 |

### 防具（10 种）

| 中文 | 类型 | 视觉关键 |
|------|------|----------|
| **圆盾** | 小盾 | 圆形带凸 |
| **皮甲** | 轻甲 | 棕色皮革 + 铆钉 |
| **生命之冠** | 头饰 | 金冠 + 红宝石（不是护身但叫"冠"） |
| **反伤甲** | 钉甲 | 全身布满外突的尖刺 |
| **重甲** | 全身板甲 | 银色钢甲 |
| **法袍** | 法师袍 | 深蓝长袍 + 兜帽 + 胸前一颗水晶眼 |
| **斗篷** | 隐匿装 | 灰黑斗篷，飘动 |
| **重铠** | 史诗甲 | 极重板甲 + 金色边纹 |
| **鳞甲** | 龙鳞甲 | 鳞片层叠、深绿/暗银 |
| **意念甲** | 玄学甲 | 没有实体，是一个**普罗维登斯之眼**形状的杏仁形光芒 |

### 史诗 / 神话装备（少量）

- **王者之剑**（♠ Excalibur）：金色长剑、剑身刻满符文
- **天命之刃**（♠ 天命）：刀身像被命运裁切过的不规则形
- **不灭之心**（♥ 重铠级胸甲）：胸口一颗发光的红色心脏
- **众神之怒**（神罚法术）：天空裂开，神罚降临
- **复读机**（玩梗装备）：旁白"时间在打嗝"

## 7. 附魔 / 符文 — 武器上的玄学层

铁匠铺给武器附魔，共 13 种。名字本身带**修真/秘法/中式神话**气息，可作为符文雕刻在武器/盔甲/卷轴上。

**单种族附魔（5 种）**：强袭、算计、收割、撼地、幻影
**复合附魔（8 种）**：战狂血誓、重甲列阵、风行步、凝神、血祭仪、守护契、秘法回响、符文护盾

视觉建议：**毛笔字 + 微微荧光**。颜色对应消耗的碎片：兽 = 黄褐、人型 = 银白、不死 = 紫黑、巨怪 = 灰岩、暗影 = 红黑。

## 8. 玩家本体

- **起手 HP 40**，初始装备**一把短剑**，**16 张牌**
- 没有固定外观——玩家是个抽象的"爬塔者"，3D 场景里可以是**第三人称背影 + 卡牌环绕**，或**根本不出现，只出现手 + 卡牌**
- 玩家的牌组是核心 UI 元素，可考虑**手牌在腰间一字排开，悬浮在空中**这种视觉

## 9. 关键氛围词（用来设调子）

游戏内出现的、能直接用作 art direction 的句子：

> "★ 新的爬塔之旅开始 ★"
> "塔影笼罩着第一片荒原，狼嚎在远处此起彼伏。"
> "潮湿的洞穴回响着尖锐的笑声，火光摇曳。"
> "古老的咒语仍在回响，骨笛吟唱不息。"
> "光明拒绝在这里停留，只剩低语。"
> "众神之怒：天空裂开，神罚降临。"
> "命运之轮在你面前转动。"
> "锁链已经锈蚀。"

整体调性：**沉、暗、有一点诡异、不血腥但压抑**，类似 Dark Souls 早期遗迹 + 黑塔的孤寂感，远离漫画式色彩。

## 10. 3D 场景制作建议优先级

如果时间有限，优先做这些（按重要度排序）：

1. **塔身大景** × 4：四个主题段各一张全景（荒野/洞窟/祭坛/禁地），用来作 loading / 楼层切换过场
2. **每层节点地图**：一个可视化的纵向塔层，节点用上面 §1 的图标和颜色
3. **5 个 NPC 场景** × 5：商人摊、赌徒桌、神龛、术士台、宝箱
4. **铁匠铺**：温热橙光 + 砧板武器悬浮
5. **Boss 战景** × 5：每个种族至少一个 Boss 房间（兽王/哥布林王/巫妖/山岳之主/无相之主）
6. **战斗背景**（普通场景）：每个主题段 1–2 个，可作为非 Boss 战的底图

不需要为每个普通敌人单独建模——它们在视觉上**通过种族 + tier 的组合**就能识别（同种族共享骨架，tier 决定装饰复杂度）。

## 11. 不要画的东西

- 任何**像素风 / 8-bit / 复古游戏风**——这游戏代码层叫 `pixel-tower` 是历史包袱，**实际视觉走的是 line-art + flat color + 微 SVG drop-shadow**，3D 应该跟这条路：**简洁几何 + 强氛围光 + 留白**，不是马里奥
- **血腥写实**——所有"血"都是符号化的（红色凹槽、血色符文），不要内脏不要解剖
- **现代元素**——没有齿轮、蒸汽、电、机械，纯魔法古典
- **饱和卡通色**——除了花色主色和节点指示色，环境保持暗调
